= 永久保留備份
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以指定備份不符合保留原則刪除的資格、以無限期保留備份。

. 若要指定不受限制地保留備份、請輸入下列命令：「mo backup update -profileprofileprofile _name｛-labelLabel [data |-archivelogs]|-ido｝-rave-unimal'
