= 檢視複本分割程序的結果
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢視所啟動的複本分割程序結果。

. 若要檢視複製分割程序的結果、請輸入下列命令：「mo clone separ-result-profileprofile [-hosthostname][-labelsepar-label |-idsepar-id][-quiet |-verbose]」


[listing]
----
smo clone split-result -profile p1 -id 8abc01ec0e78f3e2010e78f3fdd00001
----
