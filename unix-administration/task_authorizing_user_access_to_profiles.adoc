= 授權存取設定檔
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
除了角色型存取控制（RBAC）之外、SnapManager 您也可以使用此功能設定設定檔的密碼、以防止未獲授權的存取。

. 輸入下列命令：「mo認證集-profile -name profile_name [-password]」
