= 變更設定檔密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要保護儲存庫中現有的設定檔、您應該更新設定檔的密碼。使用此設定檔建立備份時、您可以套用此更新的密碼。

. 若要更新現有設定檔的設定檔密碼、請輸入下列命令：
+
'模組設定檔更新-設定檔設定檔名稱-設定檔密碼'
