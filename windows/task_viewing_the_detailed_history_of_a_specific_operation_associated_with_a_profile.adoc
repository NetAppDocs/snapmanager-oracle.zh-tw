= 檢視與設定檔相關之特定作業的詳細記錄
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢視SnapManager 與設定檔相關的特定故障操作詳細記錄。

. 若要顯示SnapManager 與設定檔相關聯之特定Selletopy作業的詳細歷史記錄資訊、請輸入下列命令：「mo history operation-show -profileprofileprofile_name｛-labelLabel |-ided｝[-quiet |-verbose]」
