= 建立系統層級的傾印檔案
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用SMO系統傾印命令來取得SnapManager 有關該主機和環境的記錄資訊。您可以指定不同的記錄層級、以收集有關特定作業、設定檔或主機與環境的資訊。

. 輸入下列命令：SMO系統傾印
+
產生的傾印

+
[listing]
----
Path:\<user-home>\Application Data\NetApp\smo\3.3.0\smo_dump_server_host.jar
----
