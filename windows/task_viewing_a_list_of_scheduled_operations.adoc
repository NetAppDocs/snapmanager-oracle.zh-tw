= 檢視排程作業清單
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢視設定檔的排程作業清單。

. 若要顯示排程作業的相關資訊、請輸入以下命令：SMO排程清單-profile profile_name [-quiet |-verbos]
