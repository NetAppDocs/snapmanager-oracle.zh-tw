= 設定目標資料庫
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
目標資料庫是Oracle資料庫、可設定為獨立式、Real Application叢集（RAC）、自動儲存管理（ASM）或任何其他支援的組合。

. 請參閱_TR-3633_來設定目標資料庫。
